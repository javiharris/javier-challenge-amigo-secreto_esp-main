Amigo Secreto

Este es un simple juego para sortear a un "Amigo Secreto". El funciinamiento es simple. Sólo debes agregar un nombre de cada participante, hacer click en añadir. Una vez agregados los participantes debes hacer click en la casilla sortear amigo y la aplicación generará un sorteo aleatorio y mostrará al amigo secreto.
